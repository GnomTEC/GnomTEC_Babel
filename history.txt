GnomTEC Babel Addon - Change History 
==========================

5.4.2.12 (2013-12-26)
============
- Improved handling of long texts which should no longer be cut off

5.4.1.11 (2013-11-26)
============
- Updated Ace3 libraries to version r1104
- Updated ChatThrottleLib to version v23
- Added additional locale files for translations to implement all supported languages of WoW
- Supported languages by author: enUS, deDE
- With google translated languages: esES, esMX, frFR, itIT, koKR, ptBR, ruRU, zhCN, zhTW
- Support translations: http://wow.curseforge.com/addons/gnomtec_babel/localization/

============
- Updated interface version for WoW update 5.4.0

5.3.0.9 (2013-06-29)
============
- Workaround added for issue with GetNumLanguages() in dropdown initialization direct after login

5.3.0.8 (2013-06-28)
============
- Changed language select button from cyclic switching to drop down.
- Adapted size of button to fit better to GnomTEC Badge toolbar
- Now the [common] prefix will correctly not be shown instead of [darnassian]
- Added user defined languages (only when prefixes are enabled)

5.3.0.7 (2013-06-21)
============
- Changed version number schema to <wow-version>.<build-number>
- Updated interface version for WoW update 5.3.0
